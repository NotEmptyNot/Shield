## Run Locally

Prerequisites:  Node.js


1. Install dependencies:
   `npm install`
2. Run the app:
   `npm run dev`

## Описание
Основой приложения является телефон, в котором необходимо пройти 5 сценариев. Изначально телефон имеет уровень безопасности 100%. При провале сценария уровень безопасности падает, если он становится 0, то игра заканчивается. В первом сценарии приходит SMS, с ссылкой для оплаты посылки, которая переводит на сайт мошенников, где они воруют данные банковской карты. Второй сценарий - в мессенджере приходит уведомление от банка с файлом, который надо срочно установить, при его скачивании, телефон захватывает вредоносная программа. Третий сценарий - в приложении банка уведомление о кэшбеке которое является безопасным. Четвертый сценарий - в ресторане поверх одного qr-кода наклеен другой, который ведет на фишинговый сайт. Пятый сценарий - qr-код с надписью о розыгрыше годового проездного на остановке, который ведет на поддельный сайт госуслуг, ворующий данные аккаунта.
